# Sistema de Gestión de Notas Estudiantiles

Este es un proyecto de consola desarrollado en Python para la gestión de calificaciones, utilizando Git como herramienta de colaboración y control de versiones.

## 👥 Integrantes y Roles
* **[Tu Nombre]** - Tech Lead (Configuración y aprobación de PRs)
* **[Nombre Integrante 2]** - Dev Backend (Lógica de registro y notas)
* **[Nombre Integrante 3]** - Dev Integración (Reportes y persistencia JSON)

## 🚀 Funcionalidades
1. **Registro de Estudiantes:** Captura de nombre e ID único.
2. **Gestión de Calificaciones:** Ingreso de notas con validación (rango 0-5).
3. **Cálculo de Promedios:** Determinación automática de estado (Aprobado/Reprobado).
4. **Reportes:** Visualización de datos en tablas formateadas por consola.
5. **Persistencia:** Guardado y carga de datos mediante archivos `.json`.
